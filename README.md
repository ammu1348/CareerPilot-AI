# 🚀 CareerPilot AI

An AI-powered Resume Analyzer that helps users evaluate their resumes, identify skills, receive personalized feedback, and explore suitable job roles.

🌐 **Live Demo:** https://careerpilot-a.netlify.app/

---

## 📌 About the Project

CareerPilot AI is a full-stack web application built to simplify resume evaluation. Users can upload their resume in PDF format, and the application analyzes it to generate:

- 📊 Resume Score
- 🛠️ Extracted Skills
- 💬 Personalized Feedback
- 💼 Recommended Job Roles
- 📄 Downloadable PDF Report

This project was built to demonstrate full-stack development, API integration, and AI-powered resume analysis.

---

## ✨ Features

- 📄 Upload Resume (PDF)
- 📊 Resume Score Visualization
- 🛠️ Automatic Skill Extraction
- 💬 AI-based Resume Feedback
- 💼 Job Recommendations
- 📄 Download Analysis Report as PDF
- 📱 Responsive UI
- ⚡ Fast API Integration
